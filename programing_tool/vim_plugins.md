# Vim 代码编辑插件

## 插件管理 Vundle
安装指南：[Vundle](https://github.com/VundleVim/Vundle.vim)

## 代码补全插件 YouCompleteMe
安装指南：[YouCompleteMe](https://github.com/Valloric/YouCompleteMe)

安装指南: [NERDTree](https://github.com/scrooloose/nerdtree)
